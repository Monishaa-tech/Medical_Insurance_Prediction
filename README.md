# 🏥 Medical Insurance Cost Prediction Using Linear Regression

## 📌 Project Description

This project predicts medical insurance charges based on customer details using the Linear Regression algorithm.

## 🎯 Objective

To estimate insurance charges using machine learning.

## 📂 Dataset

Insurance Dataset

Features:
- Age
- Gender
- BMI
- Children
- Smoker
- Region

Target:
- Charges

## 🤖 Algorithm

- Linear Regression

## 🔧 Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib

## 📊 Model Evaluation

- MAE
- MSE
- RMSE
- R² Score

## ▶️ Run the Project

Install libraries

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
